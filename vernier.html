Vernier Scales for Safe Locks Manipulation

download SVG file
(use Inkscape or convert to PDF first to print to scale)

By default, this page runs in "training" mode: you can try reading the wheel offset using the Vernier scale.

Pressing the 's' key should show the wheel offset, and pressing 's' again should draw another offset at random (and hide its value).

To generate a vernier scale for actual manipulation, uncheck the "training mode" box in the "Vernier parameters" section. You should of course set all other parameters to relevant values. The n parameter allows to get several scales in a single file, which is handy if you need more than one.
